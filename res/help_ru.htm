Введение

jBQ - это программа для чтения библейских текстов, использующая формат программы "Цитата из библии" (BibleQuote). Она поддерживает следующие типы модулей: библейские тексты, комментарии, книги, словари.

Добавление модулей

Программа jBQ поддерживает 2 способа добавления модулей: добавление внутрь JAR-файла, добавление на карту памяти устройства.

Первый способ - добавление внутрь JAR-файла

Установочный файл программы jBQ_[номер_версии].jar представляет из себя ZIP-архив, который может быть открыт в любом архиваторе, поддерживающем этот формат. Для добавления модуля из программы BibleQuote нужно скопировать папку с модулем в папку "modules" внутри JAR-файла при помощи предпочитаемого вами архиватора. Кроме того, чтобы jBQ "увидела" модуль, необходимо прописать его в файле "modules/list.txt". В этом файле каждая строка представляет из себя путь к файлу bibleqt.ini модуля относительно папки "modules" JAR-файла. Внимание: все имена файлов внутри папки "modules" JAR-файла должны быть в нижнем регистре. Для модулей-словарей последовательность аналогична с тем исключением, что в list.txt прописывается путь до .idx-файла словаря.

Добавление на карту памяти устройства

Чтобы добавить модуль на карту памяти устройства просто скопируйте папку с модулем (то есть папку, в которой находится bibleqt.ini) в папку на карту памяти, заданную в настройках как место хранения модулей (по умолчанию "bqmodules"). Модуль-словарь необходимо скопировать в подпапку Dictionaries указанной папки. Таким образом там должны оказаться .htm и .idx-файлы словаря.
